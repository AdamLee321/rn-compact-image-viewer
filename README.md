# rn-compact-image-viewer

Image viewer library to display fullscreen and compact images

## Installation


```sh
npm install rn-compact-image-viewer
```


## Usage


```js
import { multiply } from 'rn-compact-image-viewer';

// ...

const result = await multiply(3, 7);
```
